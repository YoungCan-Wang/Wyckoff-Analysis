# Changelog

## v1.7.0 (2026-03-01)

**选股更稳准、风控更聪明、推送更可靠**

- **停牌识别**：自动跳过当日数据缺失（停牌/数据延迟）的股票，不再把停牌前的旧行情误判为有效信号，避免买入复牌后连续跌停的股票
- **截面日期对齐**：个股数据与大盘数据强制对齐到同一交易日，相对强弱计算结果更准确，减少因时间轴错位产生的虚假信号
- **震仓背景校验**：Spring（终极震仓）必须发生在横盘震荡区间内，单边下跌途中的反弹不再被误认为买点
- **EVR 次日确认**：巨量滞涨信号默认等到次日验证低点未被跌破后才上报，过滤掉游资一日游和拉高出货的干扰
- **决策降级保护**：AI 给出非法止损价时，持仓单自动降级为「持有」而非拒单，不会因模型幻觉导致持仓无人看管
- **Telegram 推送支持完整格式**：推送内容加粗、代码块、链接等格式现在均可正确显示，不再出现格式符号外漏

## v1.6.0 (2026-02-27)

**稳定性与体验优化**

- 数据拉取失败时会明确展示失败的免费数据源，避免误会是程序故障
- 首页拉取在“确定数据源全失败”时不再反复重试，避免长时间卡在加载中
- 自定义导出「A股个股历史（日线）」支持多数据源自动降级，提升成功率
- 注册成功提示文案优化：无需邮件确认即可登录

## v1.5.0 (2026-02-27)

**私人决断：你的账户，单独一份买卖建议**

- 选股和研报跑完后，若你配置了持仓和 Telegram，会单独发一份「针对你账户的 Buy/Hold/Sell 建议」到你的 Telegram，只有你能看到
- 不配则自动跳过，不影响前面的选股和研报

## v1.4.0 (2026-02-27)
- 📊 **Feature**: Wyckoff Funnel 从全市场（主板 + 创业板）筛选，输出**最值得次日操作的 6 只股票**，含观察池与操作池分层。
- 🌡️ **Feature**: 根据大盘水温动态调整筛选力度，弱势市场自动提高门槛。
- 📨 **Notification**: 飞书通知支持长文自动分片发送，避免消息截断。
- 🧾 **Observability**: 模型输入落盘与日志摘要，便于核查筛选上下文。
- ⚙️ **Automation**: GitHub Actions 定时任务，支持工作日自动执行与手动触发。

## v1.3.0 (2026-02-09)
- 🤖 **Feature**: 新增「AI 分析」页面，基于多维度投委会逻辑生成深度研报（技术结构、价值质地、情绪题材、风险与交易计划等）。
- 📌 **Feature**: 支持两种分析来源——**指定股票代码**（最多 6 只）或**先筛选再分析**（在本页完成筛选，无需跳转沙里淘金页）。
- 🧭 **Feature**: AI 分析页内嵌筛选：选择战术（抗跌主力 / 突破临界 / 异常吸筹·出货 / 启动龙头）+ 股票池（手动输入或板块），执行后直接取前 6 只用于研报。
- ⏳ **UX**: 拉取数据与生成研报等待期间，随机展示股市名人名言，缓解等待感。
- 📝 **Copy**: 页面说明与筛选区块文案优化，更友好、适合直接面向用户展示。
- ✅ **体验**: AI 分析页与沙里淘金的筛选结果一致性提升，避免同一策略在不同页面表现不一致。
