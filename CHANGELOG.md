# Changelog

## v1.4.0 (2026-02-27)
- 📊 **Feature**: Wyckoff Funnel 从全市场（主板 + 创业板）筛选，输出**最值得次日操作的 6 只股票**，含观察池与操作池分层。
- 🌡️ **Feature**: 根据大盘水温动态调整筛选力度，弱势市场自动提高门槛。
- 📨 **Notification**: 飞书通知支持长文自动分片发送，避免消息截断。
- 🧾 **Observability**: 模型输入落盘与日志摘要，便于核查筛选上下文。
- ⚙️ **Automation**: GitHub Actions 定时任务，支持工作日自动执行与手动触发。

## v1.3.0 (2026-02-09)
- 🤖 **Feature**: 新增「AI 分析」页面，基于多维度投委会逻辑生成深度研报（技术结构、价值质地、情绪题材、风险与交易计划等）。
- 📌 **Feature**: 支持两种分析来源——**指定股票代码**（最多 6 只）或**先筛选再分析**（在本页完成筛选，无需跳转沙里淘金页）。
- 🧭 **Feature**: AI 分析页内嵌筛选：选择战术（抗跌主力 / 突破临界 / 异常吸筹·出货 / 启动龙头）+ 股票池（手动输入或板块），执行后直接取前 6 只用于研报。
- ⏳ **UX**: 拉取数据与生成研报等待期间，随机展示股市名人名言，缓解等待感。
- 📝 **Copy**: 页面说明与筛选区块文案优化，更友好、适合直接面向用户展示。
- 🔧 **Refactor**: 抽离沙里淘金筛选逻辑至 `wyckoff_engine`，供 AI 分析页与沙里淘金页共用。

## v1.2.1 (2026-01-19)
- 🔐 **Auth**: 引入 **Supabase Auth** 登录系统（邮箱/密码），保护用户个性化数据。
- ⚙️ **Feature**: 新增「设置」页面，支持配置飞书 Webhook 和 Gemini API Key。
- ☁️ **Sync**: 实现用户配置（Key/Webhook）**云端自动同步**，支持多端漫游。
- 🧭 **Refactor**: 重构导航栏逻辑，新增设置入口。
- 🧹 **Clean**: 移除首页冗余配置项，界面更清爽。

## v1.1.0 (2026-01-16)
- 🧰 **Feature**: 新增「自定义导出」页面：选择数据源 → 获取数据 → 字段网格勾选 → 导出 CSV（支持全选/筛选字段/导出所选或全部字段）。
- 📦 **Feature**: 新增批量生成导出（一次最多 6 个），支持 `;`/`；` 分号分隔输入，自动去重并输出 ZIP（每只股票含增强版 OHLCV + 原始 Hist Data）。
- ⏳ **UX**: 批量生成增加加载状态（spinner + 当前处理项提示 + 进度条），避免点击后无反馈。
- 🧭 **UI**: 右侧悬浮导航新增「自定义导出」入口，便于在各页面快速跳转。

## v1.0.1 (2025-01-13)
- 📱 **UI**: 适配手机浏览器（支持手动开启手机模式）。

## v1.0.0 (2025-01-12)

- ✨ **Feature**: 支持股票代码与名称双向搜索 (Code <-> Name)。

- 📝 **Docs**: 首页增加致谢 (@Hoyooyoo) 与 GitHub 仓库链接。
